# Button

Here's an example of how to use the `e-button` component:

```tsx
<e-button label="Click me!" />
```

# Customizable CSS
The e-button component provides customizable CSS variables for theming. Here are the available variables:

| CSS Variable                      | Description                              | Default Value   |
| ----------------------------- | ---------------------------------------- | --------------- |
| --e-button-padding            | Padding of the button                    | 10px 20px       |
| --e-button-margin             | Margin of the button                     | 0px             |
| --e-button-font-size          | Font size of the button                  | 16px            |
| --e-button-background-color   | Background color of the button           | none            |
| --e-button-color              | Text color of the button                 | black           |
| --e-button-border             | Border of the button                     | 1px solid black |
| --e-button-border-radius      | Border radius of the button              | 2px             |
| --e-button-background-color-hover | Background color of the button on hover | black           |
| --e-button-color-hover        | Text color of the button on hover        | white           |
| --e-button-background-color-active | Background color of the button when active | #1f1f1e    |
| --e-button-color-active       | Text color of the button when active     | white           |
